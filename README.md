# C Practice Codes

This repository contains practice codes for C programming language. Each topic includes a set of questions and their corresponding answers. The following topics are covered:

- Operators
- Control Structures - Selection Structure
- Control Structures - Iteration Structure
- Single Dimensional Arrays
- Multi-Dimensional Arrays
- Functions

Feel free to use these codes for practicing and improving your C programming skills. The questions are designed to test your understanding of the respective topics, and the answers are provided to help you validate your solutions.

## Getting Started

To get started, clone this repository to your local machine:

```bash
git clone https://github.com/SadeeshaJayaweera/c---practice-codes.git
```

## Structure

The repository is organized into folders, with each folder representing a specific topic. Inside each topic folder, you will find a set of C files corresponding to different questions. Additionally, there will be a README file within each topic folder that provides the questions and answers.

## Usage

1. Navigate to the desired topic folder.
2. Open the Word Document file on each folder to view the questions for that topic.
3. Open the corresponding C file to view the code for each question.
4. Compile and run the C code using your preferred C compiler.
